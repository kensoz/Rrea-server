### 使用

###### インストール

```bash
yarn install
```

###### 開発（NODE_ENV=development）

```bash
yarn dev
```

###### TSコンパイラ（TSC）

```bash
yarn compile
```

###### プロジェクトコンパイラ

```bash
yarn build
```

###### テスト

```bash
yarn test
```

###### プロジェクト起動（NODE_ENV=production）

```bash
yarn start
```

###### ログクリア

```bash
yarn resetLog
```
